# vbkoroa-card

Vue 3 + Vite 기반의 개인 스택 카드 페이지입니다.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deploy To GitHub Pages

Vite `base`는 `/vbkoroa-card/`로 맞춰져 있습니다.

```sh
npm run deploy
```

배포 결과는 `gh-pages` 브랜치로 올라가며, GitHub Pages 주소는 아래 기준입니다.

`https://vbkoroa.github.io/vbkoroa-card/`
